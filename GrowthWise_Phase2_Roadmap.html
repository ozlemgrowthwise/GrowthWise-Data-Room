Measurement Health Engine
Health Score Calculation
Score = 100 − (missing_days × 5) − (tracking_errors × 10) − mismatch_penalty − anomaly_penalty

QA Checks:
✓ Missing data detection (>1 day)
✓ Spike/drop anomalies (>30% threshold)
✓ Zero conversions with active spend
✓ GTM container failures
✓ Duplicate tag detection
✓ GA4 ↔ Ads platform mismatch (>20%)
✓ Session/event consistency checks

Risks & Mitigation Strategy
High Data Inconsistencies
Strong schema normalization + dual-validation QA + reconciliation dashboards
High API Failures & Rate Limits
Retry logic with exponential backoff + fallback caching + queue management
Medium Alert Noise & False Positives
Deduplication engine + configurable thresholds + baseline learning
Medium Slow Platform Adoption
Clear upgrade triggers in free tier + onboarding automation + ROI calculators
Low Attribution & Conversion Mismatch
Surfaced as health issue with diagnostics + normalization mapping table
Low Credential Expiration & Token Refresh
Automatic token refresh + proactive expiration alerts + fallback reauthentication
